# Como contribuir com o SKYRIMRP-BR

Obrigado pelo interesse! Este repositório é **apenas de divulgação** — código-fonte do launcher, servidor e scripts privados vivem em repositórios internos.

Aqui você pode contribuir com:

---

## 🐛 Reportar bugs

Use o template de issue [`bug-report`](.github/ISSUE_TEMPLATE/bug-report.md). Forneça:

- Descrição clara do problema.
- Passos para reproduzir.
- O que você esperava vs o que aconteceu.
- Screenshots / logs (mascare tokens e IDs internos!).
- Versão do launcher (canto inferior direito da janela).

**Não** poste aqui:
- Credenciais / tokens.
- IPs internos (você não deveria ter, mas se teve, NÃO compartilhe).
- Exploits não corrigidos (use o canal privado de segurança — ver [SECURITY.md](SECURITY.md)).

---

## 💡 Sugerir funcionalidades

Use o template [`feature-request`](.github/ISSUE_TEMPLATE/feature-request.md). Antes:

1. **Pesquise** issues e discussions abertas — pode já ter sido discutido.
2. **Descreva o problema** que a feature resolve, não só a solução.
3. **Considere escopo**: features grandes viram projetos de longo prazo.

---

## 🎨 Arte e conteúdo

Aceitamos contribuições de:

- **Ícones / splash arts** para o launcher (formato: PNG/SVG, 256×256 ou 512×512).
- **Textos de lore / regras de RP** (PT-BR, sem cópia de fontes pagas).
- **Tradução** de strings do SkyMP para PT-BR (sempre via PR com diff claro).

Mande PR ou anexo na issue. Créditos são dados.

---

## 📝 Documentação

Erros de digitação, links quebrados, explicações confusas no README — PRs são bem-vindos!

Para mudanças grandes (reorganizar seções, reescrever regras), **abra issue primeiro** pra discutir.

---

## ⚖️ O que NÃO aceitamos

- **PRs com código-fonte** do launcher, servidor, gamemode, API ou bridge RakNet.
- **Modificações do executável do jogo** ou do SKSE — viola termos da Bethesda.
- **Scripts de exploit / dup / bypass** — vão direto pra ação legal.
- **Conteúdo protegido por copyright** (assets de mods proprietários sem permissão, fanart pesado demais, marcas de terceiros).
- **Spam, autopromoção, links de afiliados**.

---

## 🛠️ Workflow de PR

1. **Fork** este repositório.
2. Crie branch: `git checkout -b feat/minha-contribuicao`
3. Faça commits pequenos e descritivos (prefira Conventional Commits: `docs: corrige link quebrado`).
4. Abra PR descrevendo **o quê** e **por quê**.
5. Aguarde review. PRs podem levar alguns dias.

---

## 📜 Licença das contribuições

Ao contribuir, você concorda que seu conteúdo pode ser usado sob a licença deste repo (ver [LICENSE](LICENSE)). **Não** envie conteúdo que você não tenha direito de licenciar.

---
