# 🐉 SKYRIMRP-BR

> Servidor de roleplay multiplayer de Skyrim Special Edition, em português.

Este repositório é o **ponto oficial de divulgação e comunicação** do projeto SKYRIMRP-BR. Aqui ficam apenas materiais públicos: anúncios, roadmap, regras da comunidade e documentação de como participar.

O código-fonte, launcher, scripts de servidor e assets do jogo são propriedade do time e **não são versionados aqui**. Repositórios oficiais são divulgados nos canais abaixo.

---

## 🎮 O que é o SKYRIMRP-BR?

Um servidor privado de **Skyrim Special Edition** modificado com o projeto open-source [SkyrimMP / skymp](https://github.com/skyrim-multiplayer), adaptado para a comunidade brasileira com:

- Launcher próprio com auto-update e verificação de integridade.
- Login com conta, persistência de personagem.
- Suporte a mods vanilla-friendly (lista aprovada).
- Vocalização em PT-BR (planejado).
- Servidor dedicado 24/7.

---

## 📥 Como entrar

1. **Crie sua conta** em [site oficial — link divulgado no Discord].
2. **Baixe o launcher** pelo site oficial (NÃO baixe de terceiros — segurança).
3. **Logue no launcher** com a conta criada.
4. **Clique em JOGAR** — o launcher cuida do resto (instala mods validados, conecta no servidor, atualiza o jogo).

> ⚠️ **Atenção:** O launcher oficial **só é distribuído pelo site**. Qualquer binário vindo de Discord/Telegram/4Shared é pirataria e pode conter malware.

## 🤝 Como contribuir

SKYRIMRP-BR é mantido por uma equipe pequena. Aceitamos ajuda em:

- 🎨 **Pixel art / ícones** para o launcher
- 📝 **Tradução PT-BR** de textos do jogo (com cuidado pra não violar termos da Bethesda)
- 🐛 **Reportar bugs** com repro claro (use o template de issue)
- 📚 **Documentação** de regras de RP e lore

**Não aceitamos:**
- PRs com código-fonte do launcher ou servidor (esse repo é só divulgação).
- Modificações não autorizadas do cliente — viola termos da Bethesda.
- Scripts de exploit, dup ou bypass.

Veja [CONTRIBUTING.md](CONTRIBUTING.md) para detalhes.
